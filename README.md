# BertoAI

BertoAI, Flask altyapısı kullanılarak geliştirilen yapay zekâ destekli bir dijital asistan projesidir. Kullanıcılarla doğal dil üzerinden iletişim kurabilir, soruları yanıtlayabilir ve çeşitli görevlerde yardımcı olabilir.

## Özellikler

* Yapay zekâ destekli sohbet sistemi
* Modern ve kullanıcı dostu arayüz
* Flask tabanlı web uygulaması
* Hızlı ve güvenli çalışma yapısı
* Geliştirilebilir modüler mimari

## Kullanılan Teknolojiler

* Python
* Flask
* HTML5
* CSS3
* JavaScript

## Amaç

Bu proje, yapay zekâ teknolojilerini günlük kullanıma uygun bir web platformu içerisinde sunmak ve kullanıcıların bilgiye daha hızlı erişebilmesini sağlamayı amaçlamaktadır.
